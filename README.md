# Welcome to Réseau Étudiant 👋

> Création d'une lateforme d'Information et réseau social pour les
étudiants du College Maisonneuve avec Laravel (CRUD).

## Technologies 🖥 👩‍💻
- Laravel (PHP)
- MySQL
- HTML (blade), CSS (Bootstrap), JS

## Run tests

```sh
php artisan serve
```
